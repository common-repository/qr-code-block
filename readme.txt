Create QR-Code Blocks in the Gutenberg Editor.

== Description ==

The QR-Code Block gives you the ability to create QR-Code Blocks in the Gutenberg editor.

Features:
* WYSIWYG (no coding)
* Add websites, links or coordinates
* Change the size
* Change the color
* Change the background
